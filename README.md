# 🌌 GAME XUYÊN KHÔNG: LÀM LẠI CUỘC ĐỜI 💸

> "Nếu bạn được sống lại từ đầu... bạn sẽ chọn yêu, hay chọn giàu?"

---

## 🎮 Giới thiệu

Đây là một tựa game xuyên không – lựa chọn – làm giàu nơi bạn:

- Quay về quá khứ 🕰️  
- Xây dựng lại cuộc đời từ con số 0 💼  
- Đưa ra những quyết định thay đổi số phận 🔀  
- Đối mặt với tình yêu ❤️, sự nghiệp 💼 và drama 😈  

👉 Mỗi lựa chọn = một kết cục khác nhau!

---

## ✨ Tính năng

- 🔀 Nhiều lựa chọn (choice-based)
- 💰 Hệ thống tài chính (kiếm tiền, đầu tư, tiết kiệm)
- ❤️ Tuyến tình cảm (có thể chọn hoặc bỏ)
- 🎭 Drama + plot twist
- 🎯 Nhiều ending:
  - 👑 Tỷ phú
  - 💔 Thất bại
  - 💍 Kết hôn
  - 😎 Cuộc sống chill
